┌──────────────────────────────────────────────────────────────────────────────┐
│ WHAT ERGO CAN SCAN (Default Configuration)                                  │
└──────────────────────────────────────────────────────────────────────────────┘

✅ ENABLED:
  • Process info (/proc) - names, PIDs, command lines
  • Active windows (xdotool) - titles and process names [needs X11]
  • Git repositories - status, branches, commits
  • Shell history - ~/.bash_history, ~/.zsh_history
  • File metadata - paths, modification times (NOT contents)
  • Neovim context - current file, cursor, diagnostics

❌ DISABLED:
  • Screenshots - ENABLE_SCREENSHOTS=false
  • Browser tracking - ENABLE_BROWSER_TRACKING=false
  • Voice recording - ENABLE_VOICE=false

🔒 ALWAYS BLOCKED (Privacy Filters):
  • Files: *password*, *token*, *secret*, *.env*, *vault*, *credentials*
  • Dirs: ~/.ssh, ~/.gnupg, ~/.password-store, ~/Documents/secrets

  $ curl http://127.0.0.1:8765/health

┌──────────────────────────────────────────────────────────────────────────────┐
│ EMERGENCY CONTROLS                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

Stop all scanning:
  $ pkill -f ergo-daemon

Remove sockets:
  $ rm /tmp/ergo-daemon.sock /tmp/ergo-nvim.sock

Pause without stopping (in .env):
  PRIVACY_MODE=true

Disable specific features (in .env):
  ENABLE_GIT_TRACKING=false
  ENABLE_SHELL_TRACKING=false
  ALLOW_WINDOW_MONITORING=false

┌──────────────────────────────────────────────────────────────────────────────┐
│ INSTALLATION STEPS                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

1. Install Python deps:

   $ pip install -r requirements.txt

3. Install Neovim plugin:
   See: NEOVIM_SETUP.md

   Terminal 1: $ cargo run --bin ergo-daemon

   $ ./scripts/check-permissions.sh
